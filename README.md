📌 사이트 소개

🎯 서비스 개요

본 사이트는 AI 기반 자동 채점 시스템을 제공합니다. 교수님과 교육 담당자는 문제 파일 및 학생 풀이 파일을 업로드하면, AI가 자동으로 채점을 수행하고 엑셀 파일로 결과를 반환합니다. 이를 통해 채점 시간을 절약하고, 공정한 평가가 가능하도록 돕습니다.

🏆 주요 기능

🖼 이미지 기반 채점: 이미지(PNG, JPG, JPEG, PDF)를 인식하여 자동 채점 수행

📂 압축 파일 지원: 문제 및 풀이 파일을 ZIP 형식으로 업로드 가능

✅ 부분 점수 지원: 체크박스를 통해 부분 점수 허용 여부 선택 가능

📊 채점 결과 제공: AI가 채점 후 엑셀 파일로 결과 반환

🚀 사용 방법

교수님 사번, 과목명, 과제 분류 입력, 부분 점수 허용 여부 선택
